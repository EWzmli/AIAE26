# AIAE26 云开发迁移说明

## 快速部署步骤

### 1. 开通云开发
- 打开微信开发者工具
- 点击左上角 「云开发」 按钮
- 点击 「开通」
- 选择 「按量付费」（有免费额度）
- 记录环境ID

### 2. 配置环境ID
修改 `app.js` 中的环境ID：
```javascript
wx.cloud.init({
  env: '你的环境ID',  // 替换为你的云开发环境ID
  traceUser: true
});
```

### 3. 部署云函数
在微信开发者工具中：
1. 右键 `cloudfunctions/auth` → 「创建并部署：云端安装依赖」
2. 右键 `cloudfunctions/posts` → 「创建并部署：云端安装依赖」

### 4. 创建数据库集合
进入 「云开发」 → 「数据库」，创建以下集合：
- `users` - 用户资料
- `posts` - 社区帖子
- `events` - 活动列表

设置权限为：**所有用户可读，仅创建者可写**

### 5. 上传并发布
- 点击开发者工具右上角 「上传」
- 填写版本号（如：1.0.0-cloud）
- 登录微信公众平台提交审核
