## 2026-03-05 头像系统与名片打印功能

### 新增功能

#### 1. 头像系统 (`utils/avatarGenerator.js`)
- **预设头像库**：16款风格化头像（商务/科技/创意/自然）
- **Canvas头像生成**：渐变色背景+图标组合
- **文字头像**：使用姓名首字母生成
- **相册上传**：从本地相册选择并压缩
- **上传接口**：对接后端头像上传API

#### 2. 头像选择页面 (`pages/profile/avatar-select`)
- 三标签切换：预设头像 / AI生成 / 相册上传
- 风格筛选功能
- 实时预览
- 一键保存

#### 3. 名片系统升级 (`pages/profile/card`)
- **高清名片生成**：Canvas绘制1050×600像素名片
- **专业设计模板**：
  - 渐变背景 (#1a1a2e → #16213e)
  - 左侧头像区域
  - 右侧信息区域
  - AIAE品牌标识
- **打印支持**：
  - 保存高清图片到相册
  - 打印规格说明（90mm×54mm）
  - 分享到打印店
- **操作菜单**：保存/打印/刷新/分享

#### 4. 编辑资料页面升级 (`pages/profile/edit`)
- 头像编辑入口
- 状态标签选择
- 技能/兴趣标签管理
- 表单验证

#### 5. 后端API (`routes/upload.js`)
- 头像上传接口 (POST /api/upload/avatar)
- Base64图片处理
- 文件存储管理
- 静态文件服务

### 技术亮点
1. **Canvas 2D API**：使用微信小程序2D Canvas绘制高质量图片
2. **Promise封装**：所有生成函数返回Promise，便于异步处理
3. **模块化设计**：工具函数与页面逻辑分离，便于复用
4. **渐进增强**：无头像时显示首字母，有头像时显示图片

### 使用说明
1. 进入"我的" → "编辑资料" → 点击头像 → 选择头像来源
2. 进入"我的" → "我的名片" → 查看/保存/打印名片
3. 名片建议打印规格：90mm × 54mm，300g铜版纸
